terreno_proprio :: Mapa -> Bool #

A função terreno_proprio verifica se não existem obstáculos em terrenos impróprios, como por exemplo, troncos em estradas ou relvas, árvores em rios ou estradas.

Propriedade :

terreno_proprio (Mapa l []) = True

Exemplos de utilização :

>>> terreno_proprio (Mapa 3 [(Relva,[Carro])])
False

>>> terreno_proprio (Mapa 3 [(Relva,[Nenhum,Árvore])])
True
